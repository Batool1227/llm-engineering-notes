# What is a Large Language Model (LLM)?

A Large Language Model (LLM) is a neural network trained on large amounts of text to predict the next token in a sequence. By learning statistical patterns from text, an LLM can generate coherent responses, answer questions, summarize documents, write code, and perform many other language-related tasks.

Although the underlying objective is next-token prediction, the same model can be adapted to a wide range of applications through fine-tuning, prompting, tool use, and retrieval techniques.
